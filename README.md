# Currency Converter Script

A Python script that fetches live exchange rates from the ExchangeRate API and logs them to a CSV file with currency symbols.

## Features
- Fetches live USD to NGN and USD to GBP rates
- Saves data to CSV with timestamps
- Displays currency symbols (₦, £)
- Auto-adds headers if file is new

## Requirements
- Python 3.x
- `requests` library
